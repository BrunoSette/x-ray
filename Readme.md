## Installation

```
git clone https://github.com/BrunoSette/x-ray/

npm install
```

## Running Amazon Scrapper

```
cd components

node chupaAmazon.js
```

## Running OLX Scrapper

```
cd components

node chupaCabra.js
```

The result files will be created in /results
